(static) errorIfNotTypesThrower(contextName) → {function|errorIfNotTypes}

Returns a function that can be used to ensure that a value is of one or more given types. This function is the same as `getErrorIfNotTypesThrower` except it doesn't expect an `errorMessageCall` or template function (uses a default-ly defined one) The returned function is used in cases where informative error messages containing the value types, names, expected type names, are-required/should-be-used etc.
